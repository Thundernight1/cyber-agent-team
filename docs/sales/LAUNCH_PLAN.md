## 📞 EMERGENCY TROUBLESHOOTING

**"Nobody's replying to my emails"**
→ Emails might be going to spam
→ Try: Different email (Yahoo, Gmail, new account)
→ Try: Add personal note (shows real person)
→ Try: Call instead of email

**"People aren't taking calls"**
→ Might be wrong person/wrong company
→ Try: Target smaller companies first (easier to reach)
→ Try: Call before email (warmer)
→ Try: Reference someone who recommended you

**"I don't know what to say on the call"**
→ Use script provided above
→ Practice with 3 friends first
→ Record yourself
→ Refine

**"They said no"**
→ "What would have to be true for this to make sense?"
→ Take notes
→ Move to next company
